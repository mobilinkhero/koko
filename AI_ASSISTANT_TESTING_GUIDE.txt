WhatsMark Personal AI Assistant - Complete Testing Guide
=========================================================

TESTING OVERVIEW
================

This guide provides comprehensive testing scenarios for the WhatsMark Personal AI Assistant system. Use the provided demo files to thoroughly test all AI capabilities.

TEST FILES PROVIDED
==================

1. WHATSMARK_SERVICES_DEMO.md - Main service documentation (Markdown)
2. WHATSMARK_PRICING_DEMO.csv - Pricing and plan data (CSV)
3. WHATSMARK_TECHNICAL_SPECS.json - Technical specifications (JSON)
4. AI_ASSISTANT_TESTING_GUIDE.txt - This testing guide (Text)

STEP-BY-STEP TESTING PROCESS
============================

PHASE 1: BASIC SETUP (5 minutes)
--------------------------------

1. Navigate to /{subdomain}/ai-assistant
2. Click "Create New Assistant"
3. Fill in basic information:
   - Name: "WhatsMark Demo Assistant"
   - Description: "Complete business support AI for testing"
   - Use Cases: Select all (FAQ, Product, Onboarding, CSV, SOP)

4. Configure AI settings:
   - Model: GPT-4o Mini (recommended for testing)
   - Temperature: 0.7 (balanced creativity)
   - Max Tokens: 1000 (good for detailed responses)

5. Add system instructions:
   "You are a helpful WhatsMark customer support assistant. Use the uploaded documents to provide accurate information about our services, pricing, and technical specifications. Always be professional and helpful."

PHASE 2: DOCUMENT UPLOAD (10 minutes)
-------------------------------------

1. Upload all test files:
   - Drag and drop WHATSMARK_SERVICES_DEMO.md
   - Upload WHATSMARK_PRICING_DEMO.csv
   - Add WHATSMARK_TECHNICAL_SPECS.json
   - Include AI_ASSISTANT_TESTING_GUIDE.txt

2. Wait for processing (should take 30-60 seconds)

3. Verify uploads:
   - Check file count shows 4 files
   - Verify total character count is displayed
   - Ensure no error messages appear

PERFORMANCE BENCHMARKS
======================

Response Time Targets:
- Simple queries (info lookup): < 3 seconds
- Complex queries (reasoning): < 8 seconds
- CSV lookups: < 5 seconds
- JSON data retrieval: < 4 seconds

Accuracy Targets:
- Factual information: 95%+ accuracy
- Pricing data: 100% accuracy
- Technical specs: 100% accuracy
- Business recommendations: 85%+ relevance

COMMON ISSUES & TROUBLESHOOTING
===============================

Issue: "AI doesn't recognize uploaded files"
Solution: Check file processing status, ensure files are under 5MB, verify supported formats.

Issue: "Responses are too generic"
Solution: Improve system instructions, add more specific business context, adjust temperature.

Issue: "AI can't find CSV data"
Solution: Verify CSV structure, check for proper headers, ensure data format is clear.

Issue: "Responses are too long/short"
Solution: Adjust max tokens setting, refine system instructions with length preferences.
